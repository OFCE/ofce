# Palette de couleurs associée de type sequentiel qui à partir de deux codes HEX génère une palette de n couleur compatible avec les normes graphiques de la revue OFCE

Palette de couleurs associée de type sequentiel qui à partir de deux
codes HEX génère une palette de n couleur compatible avec les normes
graphiques de la revue OFCE

## Usage

``` r
ofce_palette_seq(colors = c("#005DA4", "#C51315"), n = NULL)
```

## Arguments

- colors:

  vecteur de deux couleurs à partir duquel la palette est générée

- n:

  integer, nombre de couleurs

## Value

une palette de couleurs (format HEX)
