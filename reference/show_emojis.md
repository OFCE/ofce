# Afficher la liste des emojis Twemoji

Ouvre dans le navigateur la page de référence des emojis Twemoji avec
leurs codes hexadécimaux, utile pour trouver les codes à passer à
[`get_icons()`](https://ofce.github.io/ofce/reference/get_icons.md).

## Usage

``` r
show_emojis()
```

## Value

Appelée pour son effet de bord (ouverture du navigateur). Retourne
`NULL` de manière invisible.

## Examples

``` r
if (FALSE) { # \dontrun{
show_emojis()
} # }
```
