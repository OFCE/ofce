# Enregistre un tableau

L'enregistrement est fait dans le dossier spécifié dans les options

## Usage

``` r
save_gt(object)
```

## Arguments

- object:

  le ggplot

## Value

le graphique, avec un effet de bord qui est le gt enregistré
