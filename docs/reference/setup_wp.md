# installe un squelette de document de travail

prépare un dossier, avec un exemple et les extensions nécessaires pour
le formattage OFCE

## Usage

``` r
setup_wp(dir = NULL, nom = NULL)
```

## Arguments

- dir:

  Répertoire à créer. Par défaut, le qmd est placé dans le dossier de
  travail.

- nom:

  Nom du projet, "wp" par défaut,
