# Modifie la justification des textes en HTML

Attention cette fonction est un hack en attendant mieux Elle est assez
fragile et peut ne pas fonctionner

## Usage

``` r
set_justify(path = ".", justify = TRUE, ext = "ofce")
```

## Arguments

- path:

  chemin vers le ou les documents HTML

- justify:

  (boolean) TRUE pour justifier, FALSE pour aligner à gauche

- ext:

  (string) extension des fichiers à modifier (ofce par défaut)
