# Thème OFCE pour les tableaux pour gt

Applique le theme ofce compatible avec la norme de la Revue de l'OFCE

## Usage

``` r
table_ofce(data, ...)
```

## Arguments

- data:

  un objet gt

- base_family:

  character(1) string, police de charactère du thème (Stone sans par
  défaut)

## Value

un objet gt, décoré
