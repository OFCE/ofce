# Board avec Azure

Lit les dentifiants dans .Renviron et les utilise pour connecter avec
Azure. Les identifiants doivent être azure_url et un azure_jeton un
jeton sas à obtenir sur le portail azure.

## Usage

``` r
board()
```

## Value

un board (de `{pins}`)

## Details

Cette fonction permet d'utiliser les fonctions de `{pins}`.
