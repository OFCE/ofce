# Thème OFCE pour les tableaux pour gt

Applique le theme ofce compatible avec la norme de la Revue de l'OFCE

## Usage

``` r
theme.gt_ofce_2(
  data,
  largeur_colonne = NULL,
  col_highlight = NULL,
  row_highlight = NULL,
  ...
)
```

## Arguments

- data:

  un objet gt

- largeur_colonne:

  largeur de colonne (en px)

- ...:

  autres arguments à passer dans la fonction

## Value

un objet gt, décoré
