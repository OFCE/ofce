# make filename

make filename

## Usage

``` r
make_filename(x, file = "", rep = "", env, ext)
```

## Arguments

- x:

  nom de l'objet

- file:

  nom du fichier crée (par défaut nom de x)

- rep:

  stringr

- env:

  environnement

- ext:

  nom de l'extension du fichier (.png, .jpg,...)

## Value

un nom de fichier
