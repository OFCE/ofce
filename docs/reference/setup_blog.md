# installe un squelette pour le blog

prépare un dossier, avec un exemple et les extensions nécessaires pour
le formattage OFCE

## Usage

``` r
setup_blog(dir = NULL, nom = NULL)
```

## Arguments

- dir:

  Répertoire à créer. Répertoire à créer. Par défaut, le qmd est placé
  dans le dossier de travail.

- nom:

  Nom du projet, "blog" par défaut,
