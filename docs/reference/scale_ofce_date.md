# Echelle pour les x du format habituel des grahiques à date

Echelle pour les x du format habituel des grahiques à date

## Usage

``` r
scale_ofce_date(
  date_breaks = "5 years",
  labels = NULL,
  date_minor_breaks = "1 year",
  date_labels = "%Y",
  guide = "minor_breaks",
  name = NULL,
  expand = expansion(mult = c(0.03, 0.03)),
  ...
)
```

## Arguments

- date:

  liste de date

- freq:

  soit day, month, quarter, year

- lang:

  langue de retour

## Value

chaine de charactères
